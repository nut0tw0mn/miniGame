# MiniGame

โปรเจกต์นี้เป็นมินิเกมที่ทำด้วย Unity ประกอบด้วยเกมย่อยดังนี้:

## รายการมินิเกม

- **TicTacToe**
  - เกม XO 3x3 ผลัดกันลงระหว่างผู้เล่นสองคน (X และ O)
  - มีระบบตรวจผู้ชนะ และแจ้งผลเมื่อมีคนชนะหรือเสมอ

- **Memory**
  - เกมจับคู่ภาพ (Memory Matching)
  - ผู้เล่นต้องเปิดการ์ดทีละ 2 ใบให้รูปตรงกัน
  - เมื่อจับคู่ถูก การ์ดจะถูกล็อกไว้ จนกว่าจะหาได้ครบทุกคู่

- **Snake**
  - เกมงู ควบลคุมทิศทางของหัวด้วย Key W,A,S,D บนคีบอร์ด
